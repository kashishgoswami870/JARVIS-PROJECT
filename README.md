# Jarvis Voice Assistant 🤖

A Python-based voice assistant that can perform tasks based on user voice commands.

## 🚀 Features

* 🎤 Voice command recognition
* 🔊 Text-to-speech response
* 📂 Opens applications and websites
* 🌐 Performs basic search tasks

## 🛠️ Tech Used

* Python
* SpeechRecognition
* pyttsx3
* os module

## ▶️ How to Run

1. Install required libraries:

   ```
   pip install speechrecognition pyttsx3 pyaudio
   ```
2. Run the script:

   ```
   python main.py
   ```
